Laravel 常用扩展包

2018-03-05 • 标签: laravel
扩展包 描述
aws/aws-sdk-php-laravel AWS 官方提供的 SDK 扩展包, 使用它可以非常方便的操作 Amazon 的 S3、Amazon Glacier 和 Amazon DynamoDB 等资源
intervention/image 强大的图片处理扩展包，支持裁剪、水印等处理
SocialiteProviders/manager 最全的三方登录认证 http://socialiteproviders.github.io/
jenssegers/agent 客户端识别包
elasticquent/elasticquent elasticsearch model包
baum/baum 嵌套集合模型网易跟帖
venturecraft/revisionable 为 model 添加操作历史
dingo/api restful api 管理工具
jlapp/swaggervel Swagger API 规范支持
unisharp/laravel-ckeditor ckeditor 加载包
barryvdh/laravel-ide-helper IDE 开发助手
barryvdh/laravel-debugbar debugbar调试工具栏
yajra/laravel-datatables-oracle jQuery DataTables 的后端支持
lokielse/omnipay-wechatpay 微信支付
omnipay/paypal paypal 支付
ignited/laravel-omnipay 各种支付方式
arcanedev/log-viewer 日志查看
peinhu/aetherupload-laravel 大文件上传,断点续传，进度显示
aporat/store-receipt-validator appstore receipt
khill/lavacharts 图表展示
barryvdh/laravel-cors cors 跨域
darkaonline/l5-swagger swagger 工具
torann/geoip 根据IP地址确定网站访问者的地理位置
toin0u/geotools-laravel geo
lokielse/omnipay-alipay alipay 支付
fideloper/proxy
gregwar/captcha 验证码
overtrue/easy-sms 短信验证码
laracasts/generators generators 升级版，方便本地开发
